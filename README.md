# Decluter
Trata-se de um site/web voltado ao estudante moderno, que une ferramentas de organização (como listas de tarefas e calendários) a recursos de comunicação. A plataforma visa reduzir a divisão de ferramentas (como usar um app para cada função), oferecendo um ambiente único onde a colaboração por voz facilita a conclusão conjunta de atividades.
